## **Internship Tasks**

As part of my **DevelopersHub Corporation Data Science & Analytics Internship**, I have successfully completed the following tasks, gaining hands-on experience in data exploration, visualization, and predictive modeling:

---

### **Task 1: Exploring and Visualizing a Simple Dataset (Iris Dataset)**

- Loaded and explored the dataset using **pandas**.  
- Summarized the data structure and content using `.shape`, `.columns`, and `.head()`.  
- Visualized relationships between features using **scatter plots, histograms, and box plots** with **matplotlib** and **seaborn**.  
- Learned core skills in **data inspection and visualization**.

---

### **Task 2: Credit Risk Prediction**

- Predicted whether a loan applicant would default using **Loan Prediction Dataset**.  
- Handled missing data and explored features like loan amount, education, and income.  
- Trained a **Logistic Regression / Decision Tree** classifier.  
- Evaluated model performance with **accuracy** and **confusion matrix**, practicing **binary classification and EDA**.

---

### **Task 3: Customer Churn Prediction (Bank Customers)**

- Identified customers likely to leave using **Churn Modelling Dataset**.  
- Encoded categorical features (`Geography`, `Gender`) with **Label Encoding** and **One-Hot Encoding**.  
- Trained a **Decision Tree Classifier** and analyzed **feature importance**.  
- Learned about **classification modeling and interpreting key factors influencing churn**.

---

### **Task 4: Predicting Insurance Claim Amounts**

- Estimated medical insurance charges using **Medical Cost Personal Dataset**.  
- Trained a **Linear Regression** model to predict `charges`.  
- Visualized the effects of **BMI, age, and smoking status** on insurance charges.  
- Evaluated performance using **MAE** and **RMSE**, gaining skills in **regression modeling and error analysis**.

---

### **Task 5: Personal Loan Acceptance Prediction**

- Predicted which customers are likely to accept a personal loan using **Bank Marketing Dataset**.  
- Explored features like **age, job, and marital status** with EDA and visualizations.  
- Trained a **Logistic Regression / Decision Tree** classifier.  
- Achieved **accuracy 81%** and **F1-score ~0.81**.  
- Derived business insights to identify high-probability customer segments for targeted marketing campaigns.

---
